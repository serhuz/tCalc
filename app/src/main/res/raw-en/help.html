Data input

Following input formats are supported:

  • HH:MM:SS
  • HH:MM
  • MM:SS.sss
Where H - hours, M - minutes, S - seconds, s - milliseconds.
You can also input milliseconds (3 signs after dot) when you enter data in HH:MM:SS format.

You can also input decimal numbers. All negative values must be enclosed in parentheses. Please note, that currently negative time units input is not supported.

Examples

✔ 00:10
✔ 10:24 - HH:MM
✔ 10:24.005 - MM:SS.sss
✔ 02:00:30
✔ 0:10:40.5
✘ 10:30.5
✘ :4:05
✔ 8 - number
✔ (-8) - number
✘ -4

Operations

Currently following operations are supported:

  • Addition
  • Subtraction
  • Multiplication
  • Division
Some operations with time units Т and numbers V will cause errors. These operations are:
  • Multiplication Т*Т
  • Division V/Т
  • Addition V+Т, T+V
  • Subtraction V-Т, T-V
To group operations, use parentheses().

Examples

✔ 00:10*2+01:00
✔ 00:35/5-00:01
✘ 01:00/01:00:30
✘ 00:10*(2+01:00)

Output formats

Following output formats are supported:

  • DDd. HH:MM:SS
  • HH:MM:SS
  • MM:SS
  • [HH:MM:SS]mod24
Where D - days, H - hours, M - minutes, S - seconds.
Format [HH:MM:SS]mod24 should be used for time of day calculations.
