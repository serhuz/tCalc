Ввод данных

Поддерживаются следующие форматы ввода данных:

  • ЧЧ:ММ:СС
  • ЧЧ:ММ
  • ММ:СС.ссс
Где Ч - часы, М - минуты, С - секунды, с - миллисекунды.
При вводе данных в формате ЧЧ:ММ:СС можно вводить миллисекунды (3 знака после точки).

Также поддерживается ввод десятичных чисел. При вводе отрицательного значения его необходимо заключить в скобки. В настоящее время ввод отрицательных значений поддерживается только для десятичных чисел.

Примеры

✔ 00:10
✔ 10:24 - ЧЧ:ММ
✔ 10:24.005 - ММ:СС.сс
✔ 02:00:30
✔ 0:10:40.5
✘ 10:30.5
✘ :4:05
✔ 8 - число
✔ (-8) - число
✘ -4

Операции

На данный момент поддерживаются следующие операции:

  • Сложение
  • Вычитание
  • Умножение
  • Деление
Некоторые операции над единицами времени Т и значениями V вызовут ошибку. К ним относятся:
  • Умножение Т*Т
  • Деление V/Т
  • Сложение V+Т, T+V
  • Вычитание V-Т, T-V
Для группировки операций используйте скобки ().

Примеры

✔ 00:10*2+01:00
✔ 00:35/5-00:01
✘ 01:00/01:00:30
✘ 00:10*(2+01:00)

Форматы вывода

Поддерживаются следующие форматы вывода данных:

  • ДДd. ЧЧ:ММ:СС
  • ЧЧ:ММ:СС
  • ММ:СС
  • [ЧЧ:ММ:СС]mod24
Где Д - дни, Ч - часы, М - минуты, С - секунды.
Формат [ЧЧ:ММ:СС]mod24 следует использовать для вычисления времени суток при переходе дат.
